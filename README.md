## If you want to make your own themes here is a quick tutorial

<small>After this you should download and instal this Twlight menu Add-ons:</small>

> <small>**Better DSi Menu Music:** Increases the quality of the music from 16khz to 32khz, along with extending the length from 49 seconds to 8 minutes and 39 seconds to ensure no parts have been left out.</small>
>
> <small>**Extra UI Music:** Adds more music to the DSi, 3DS, and HBL UIs (3DS HOME Menu music, DSi Shop music, and HBL music).</small>
>
> https://ds-homebrew.com

1. <small>**Download a template:** Like the basic white ds theme: https://ds-homebrew.com</small>
2. <small>**Edit with PAINT:** Go to the folders and open the pngs with PAINT. No Gimp or Photoshop, PAINT is perfect for doing this type of things.</small>
3. <small>**Resolution:** The canvas resolution has to be **256x192**. Make sure the canvas size is that or your theme wont work.</small>
4. <small>**Design:** Edit the photos (don´t touch the navigation bar atelast you could make other better, also don´t touch other elements that seems important, like the boxart container).</small>
5. <small>**Save:** Save the changes and make sure the final canvas size is **256x192**.</small>

### For custom sounds:
<small>(This is custom bg music. Making custom sounds is much more complex and i don´t recomend to try it in your first theme. More info: https://ds-homebrew.com)</small>

1. <small>Go to: https://online-convert.com</small>
2. <small>**Convert your mp3 to wav with this configurations:** Bit depth: 16 Bit | Audio frequency: 24000 Hz | Audio channels: Mono.</small>
3. <small>After converting, go to the sound folder on your theme and paste it as `bgm.wav`</small>
4. <small>In twlight menu setings change the bg music configuration to **theme custom**.</small>

### For custom quick menu:
<small>Most of themes have the quick menu folder. If your theme has it just modify the images, if not, create a folder named: `quickmenu`</small>

<small>Create 3 pngs (256x192) and name them:</small>
- <small>`topbg` (Top)</small>
- <small>`phat_topbg` (Top)</small>
- <small>`bottombg` (Bottom)</small>
